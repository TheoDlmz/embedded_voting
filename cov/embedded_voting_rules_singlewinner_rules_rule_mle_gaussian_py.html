9class RuleMLEGaussian(Rule): 

10 """ 

11 A rule that computes the scores of the candidates, assuming that the embeddings of the voters correspond to a 

12 covariance matrix. 

13 

14 For this rule, the embeddings must be a matrix `n_voters` * `n_voters`. 

15 

16 Examples 

17 -------- 

18 Consider a generating epistemic model, where the true value of each candidate is uniformly drawn in a given 

19 interval, and where the voters add a noise which is multivariate Gaussian. 

20 

21 >>> np.random.seed(42) 

22 >>> covariance_matrix = np.array([ 

23 ... [2.02, 1.96, 0.86, 0.81, 1.67], 

24 ... [1.96, 3.01, 1.46, 0.69, 1.59], 

25 ... [0.86, 1.46, 0.94, 0.39, 0.7 ], 

26 ... [0.81, 0.69, 0.39, 0.51, 0.9 ], 

27 ... [1.67, 1.59, 0.7 , 0.9 , 1.78] 

28 ... ]) 

29 >>> ratings_generator = RatingsGeneratorEpistemicMultivariate(covariance_matrix=covariance_matrix) 

30 >>> ratings = ratings_generator(n_candidates=2) 

31 >>> ratings_generator.ground_truth_ 

32 array([17.73956049, 14.3887844 ]) 

33 >>> ratings 

34 Ratings([[17.56232759, 14.51592899], 

35 [16.82544972, 15.78818081], 

36 [17.51952581, 14.44449175], 

37 [17.34964888, 14.4010885 ], 

38 [16.69480298, 14.9281998 ]]) 

39 

40 If we know the covariance matrix of the noises, then `RuleMLEGaussian` is the maximum likelihood 

41 estimator of the ground truth: 

42 

43 >>> election = RuleMLEGaussian()(ratings, embeddings=covariance_matrix) 

44 >>> election.scores_ 

45 [268.6683142769278, 221.50830758813055] 

46 

47 #>>> np.linalg.norm(ratings_generator.ground_truth_ - election.scores_) # Error estimation 

48 #0.4783006898563199 

49 

50 As a baseline, consider the error for the naive arithmetic mean: 

51 

52 #>>> scores_average = np.mean(ratings, axis=0) 

53 #>>> np.linalg.norm(ratings_generator.ground_truth_ - scores_average, 2) 

54 0.6911799682033576 

55 

56 However, in practice, we often do not know the covariance matrix of the noise. A workaround can be to 

57 use the covariance matrix of the ratings: 

58 

59 #>>> embeddings = EmbeddingsFromRatingsCovariance()(ratings) 

60 #>>> election = RuleMLEGaussian()(ratings, embeddings) 

61 #>>> election.scores_ # doctest: +ELLIPSIS 

62 #[12.90546983325..., 19.502265626617...] 

63 

64 Actually, this is the default behavior of `RuleMLEGaussian` when no embeddings are given: 

65 

66 #>>> election = RuleMLEGaussian()(ratings) 

67 #>>> election.scores_ # doctest: +ELLIPSIS 

68 #[12.90546983325..., 19.502265626617...] 

69 

70 Unfortunately, this approximation is relevant if there are a large number of candidates (to have a good estimation 

71 of the covariance matrix) and if the noise is large compared to the differences between true values (so 

72 that the covariance of ratings approximates well the covariance of noises), which is not a common case. In our 

73 example, the assumptions are not met, and the result is not even as good as the naive arithmetic mean: 

74 

75 #>>> np.linalg.norm(ratings_generator.ground_truth_ - election.scores_) # Error estimation 

76 #0.839945516610... 

77 """

80 self.tol = tol

85 @cached_property 

86 def pinv_covariance_(self): 

87 tol = self.tol 

88 n, m = self.embeddings_.shape 

89 min_d = min(n, m) 

90 u, s, v = np.linalg.svd(self.embeddings_) 

91 clean_zeros(s, tol=tol) 

92 dia = np.zeros((m, n)) 

93 dia[:min_d, :min_d] = np.diag([pseudo_inverse_scalar(e) for e in s]) 

94 inverse = v.T @ dia @ u.T 

95 clean_zeros(inverse, tol=tol) 

96 return inverse 

97 

98 @cached_property 

99 def weights_(self): 

100 return self.pinv_covariance_.sum(axis=0) 

101 

102 def _score_(self, candidate): 

103 return float(self.ratings_.candidate_ratings(candidate) @ self.weights_)
