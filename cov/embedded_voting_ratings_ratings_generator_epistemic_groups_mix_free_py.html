6class RatingsGeneratorEpistemicGroupsMixFree(RatingsGeneratorEpistemicGroups): 

7 """ 

8 A generator of ratings such that voters are 

9 separated into different groups and the noise of 

10 an voter on a candidate is equal to the noise 

11 of his group plus his own independent noise. 

12 The noise of different groups can be correlated due 

13 to the group features. 

14 

15 For each candidate `i`: 

16 

17 * For each feature, a `sigma_feature` is drawn (absolute part of a normal variable, scaled by 

18 `group_noise`). Then a `noise_feature` is drawn according to `group_noise_f` (scaled by `group_noise`). 

19 * For each group, `noise_group` is the barycenter of the values of `noise_feature`, with the 

20 weights for each feature given by `groups_features`. 

21 * For each voter, `noise_dependent` is equal to the `noise_group` of her group. 

22 * For each voter, `noise_independent` is drawn according to `independent_noise_f` (scaled by `independent_noise`). 

23 * For each voter of each group, the rating is computed as 

24 `ground_truth[i] + noise_dependent + noise_independent`. 

25 

26 Parameters 

27 ---------- 

28 groups_sizes : list or np.ndarray 

29 The number of voters in each groups. 

30 The sum is equal to :attr:`~embedded_voting.RatingsGenerator.n_voters`. 

31 groups_features : list or np.ndarray 

32 The features of each group of voters. 

33 Should be of the same length than :attr:`group_sizes`. 

34 Each row of this matrix correspond to the features of a group. 

35 group_noise : float 

36 The variance used to sample the noise of each group. 

37 independent_noise : float 

38 The variance used to sample the independent noise of each voter. 

39 truth_generator : TruthGenerator 

40 The truth generator used to generate to true values of each candidate. 

41 Default: `TruthGeneratorUniform(10, 20)`. 

42 group_noise_f : function 

43 The function used to sample the noise of each group. 

44 Default: `np.random.normal`. 

45 independent_noise_f : function 

46 The function used to sample the independent noise of each voter. 

47 Default: `np.random.normal`. 

48 

49 Attributes 

50 ---------- 

51 ground_truth_ : np.ndarray 

52 The ground truth ("true value") for each candidate, corresponding to the 

53 last ratings generated. 

54 

55 Examples 

56 -------- 

57 >>> np.random.seed(42) 

58 >>> features = [[1, 0], [0, 1], [1, 1]] 

59 >>> generator = RatingsGeneratorEpistemicGroupsMixFree([2, 2, 2], features, group_noise_f=np.random.normal, independent_noise_f=np.random.normal) 

60 >>> generator() # doctest: +ELLIPSIS

62 [18.23627...], 

63 [17.60129...], 

64 [17.60129...], 

65 [17.99302...], 

66 [17.99302...]]) 

67 >>> generator.ground_truth_ # doctest: +ELLIPSIS 

68 array([17.73956...]) 

69 """
